* Overview

Scientific Page View provides a word processor-like interface for writing scientific papers in org-mode, organized chronologically following established academic writing guidelines from official sources (PLOS, Nature, Science, ICMJE, university thesis requirements).

* Chronological Writing Process

The system enforces the official academic writing process order:

**Phase 1: Preparation and Planning**
- ~scientific-page-view-start-writing-session()~ - Initialize session
- Set paper type and journal requirements
- Display word count limits from official standards
- Configure page layout settings

**Phase 2: Literature Review**
- Use ~scientific-page-view-insert-section-template("Literature Review")~
- Follow systematic review structure
- Track word counts against standards
- Document search strategy

**Phase 3: Methods and Design**
- Use ~scientific-page-view-insert-section-template("Methods")~
- Structure research design clearly
- Include ethical considerations
- Add participant/sample descriptions

**Phase 4: Results and Data Presentation**
- Use ~scientific-page-view-insert-section-template("Results")~
- Format tables with proper captions
- Insert figures with placeholders
- Present statistical findings

**Phase 5: Discussion and Analysis**
- Use ~scientific-page-view-insert-section-template("Discussion")~
- Interpret findings in context
- Compare with literature
- Discuss limitations

**Phase 6: Conclusion and Implications**
- Use ~scientific-page-view-insert-section-template("Conclusion")~
- Summarize key findings
- State conclusions clearly
- Highlight contributions

* Academic Standards Compliance

**Official Sources Used**

This system implements standards from:

- *PLOS (Public Library of Science)*:
  - Abstract word count: 250 words
  - IMRaD structure enforced
  - Reference formatting requirements

- *Nature / Science*:
  - 6-page maximum limit
  - Abstract: 150 words
  - Methods and Results sections

- *ICMJE (International Committee of Medical Journal Editors)*:
  - Structured abstract format
  - Reporting guidelines
  - Table and figure standards

- *University Thesis Requirements*:
  - Master's thesis: 40-80 pages
  - Doctoral dissertation: 150-300+ pages
  - IRB approval placeholders

* Page Layout Features

- *Font*: Times New Roman, 12pt (academic standard)
- *Spacing*: 1.5 line spacing (standard for readability)

- *Headers*: Document title, author, date at top of each page

* Troubleshooting

**Page Breaks Not Appearing**

*Problem*: Page breaks not showing after inserting
*Solution*:
#+BEGIN_SRC elisp
;; Ensure olivetti-mode is active
(olivetti-mode 1)

;; Re-apply page breaks
M-x scientific-page-view-clear
M-x scientific-page-view-initialize
#+END_SRC

**Word Count Warnings**

*Problem*: Seeing word count warnings constantly
*Solution*:
#+BEGIN_SRC elisp
;; Adjust limits or disable enforcement
(setq scientific-page-view-enforce-word-counts nil)
#+END_SRC

**Citation Formatting Issues**

*Problem*: Citations not matching journal style
*Solution*:
#+BEGIN_SRC elisp

(setq scientific-page-view-citation-style 'ieee)

(setq scientific-page-view-journal-type 'ieee)
#+END_SRC

* Best Practices

**Academic Writing Workflow**

1. *Plan Before Writing*
   - Use ~scientific-page-view-start-writing-session~ to set goals
   - Review word count limits for your paper type
   - Choose appropriate journal type

2. *Follow Chronological Structure*
   - Write sections in order: Abstract → Intro → Methods → Results → Discussion → Conclusion
   - Use page breaks between major sections
   - Add page numbers for navigation

3. *Use Templates When Possible*
   - Start with ~M-x yas-new-snippet~
   - Select appropriate template for your paper type
   - Templates ensure proper structure

4. *Check Word Counts Regularly*
   - Use ~M-x scientific-page-view-show-word-counts~
   - Address sections exceeding limits
   - Focus on conciseness and clarity

5. *Validate Before Submission*
   - Use ~M-x scientific-page-view-check-references~
   - Ensure all citations properly formatted
   - Check figure and table captions
   - Verify abstract meets word limit

6. *Use Debug Mode for Layout*
   - ~M-x scientific-page-view-toggle-debug~ to see page structure
   - Verify page breaks are in correct positions
   - Check line counts per page

**Citation Management**

1. *Choose Style Early*: Set citation style before writing
2. *Be Consistent*: Use same format throughout
3. *Format as You Write*: Use formatting functions during writing
4. *Validate at End*: Check all citations match style

**Figure and Table Management**

1. *Use Placeholders*: Insert figures with caption placeholders first
2. *Fill Captions Later*: Complete descriptions after data is ready
3. *Auto-Number*: Use ~M-x scientific-page-view-number-figures~ when figures complete
4. *Keep Captions Descriptive*: Explain what figure/table shows in caption
